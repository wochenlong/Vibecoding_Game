# GBIT Monster Quest

> 一款由 AI 全程协作开发的宝可梦风格怪兽 RPG，内嵌 AI 生图管线，从代码到美术均通过自然语言驱动生成。

## 这是什么游戏

你扮演一位刚踏上旅程的训练师，在晨曦镇领取初始怪兽伙伴，穿越花冠大道捕捉野生怪兽，挑战道馆馆主赢得第一枚徽章。

核心玩法：

- 自由探索地图、触发野外遭遇
- 回合制战斗，支持技能、捕捉、道具、逃跑
- 怪兽升级、进化、技能替换
- 45 种原创怪兽、10 种属性克制关系
- 每只怪兽拥有随机个体值（IV），同种不同只有细微属性差异
- 被动特性系统（Trait），影响战斗节奏与策略
- 全程自动存档，随时关闭随时继续

## 创新点

### AI 协作开发

- **100% 自然语言驱动**：所有游戏代码、数值设计、平衡配置均通过与 AI（Claude / Cursor）对话生成，开发者不手写任何一行核心逻辑
- **可调数值与代码解耦**：通过 `config/balance-config.json` 外置配置，AI 只需修改 JSON 即可调整游戏平衡，不动核心 JS
- **AI 生图管线**：怪兽立绘、NPC 形象、场景底图均通过 AI 接口按需生成并自动注册到游戏资源清单

### 游戏设计

- **个体值（IV）系统**：每只捕获的怪兽各属性有 0–15 的随机差异，带来"好个体"追求空间
- **特性（Trait）系统**：每种怪兽携带独特被动特性（如"岩盾""疾风""灼热"），改变战斗中的伤害/速度/命中计算
- **帧式过场动画**：序幕与关键剧情节点采用双屏台词 + 渐变切场演出，可跳过
- **技能学习面板**：升级/进化后弹出 4 选 1 技能替换界面，还原主系游戏核心决策体验

## 核心做法

1. **世界观先行**：用 AI 生成完整世界设定（地图、种族、属性关系），所有后续设计以此为基准
2. **模块化架构**：战斗 / 世界 / UI / 渲染 / 平衡配置各自独立模块，AI 每次只改一个模块，降低引入 Bug 的风险
3. **验收驱动**：每个功能完成后用 Playwright 截图脚本自动验收，确保改动没有破坏已有流程
4. **增量平衡**：通过 `balance-config.json` 的 offset 机制，AI 每次只提交数值增量，可随时回滚
